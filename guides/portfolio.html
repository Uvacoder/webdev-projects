Portfolio Project Guide

Day 1

  1. Create a new html document with a DOCTYPE and a title
  2. Add a main content area, centered horizontally on the page
  3. Add several sections of content using h1, h2, h3, and p, with css styles
  4. Use strong or em to style text within a paragraph
  5. Add a nav element at the top of the page with a link to each heading in your main section
  6. Float an img to the right of your main content
  7. Test your UI on mobile and add at least one @media query for small screens
  8. Ensure your HTML is valid using https://validator.w3.org/

Day 2

  1. Create a form using display: flex for layout
  2. Add a text input with a label
  3. Add a select menu with a label
  4. Add a textarea with a label
  5. Test your UI on mobile and add at least one @media query for small screens
  6. Support submitting the form to https://echo-form-body-u6ozq7lkydx0.runkit.sh/
  7. Enable VoiceOver and navigate through your page. Identify (and possibly improve) any elements that VoiceOver doesn't handle well
  8. Ensure your HTML is valid using https://validator.w3.org/
