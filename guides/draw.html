Draw Project Guide

Day 1

  1. Download the starter code.
  2. Create an HTML, CSS, and TypeScript file. Connect these files together so that a console.log displays in the JavaScript console.
  3. In your HTML document, create a canvas element and a radio buttons for the different tools: rectangle, circle, and pencil
  4. Write a function to read the currently selected radio button from the DOM. This function should return an enum or union type.
  5. Define types for Point, Rect, RectangleShape, EllipseShape, and PolylineShape.
  6. Hardcode an example rectangle, ellipse, and polyline in a new array.
  7. Write a Drawing class. This class should have a draw method or property that draws an array of shapes onto the canvas. Start by drawing the hardcoded array.
  8. Call your draw method on a timer, so that if the array of shapes changes, it's reflected on the canvas.
